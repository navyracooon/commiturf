# Commiturf Privacy Policy

Last updated: July 23, 2026

Commiturf turns contribution activity already visible on a public GitHub profile into a visual garden. Commiturf is an independent app and is not affiliated with GitHub.

## Data used by the app

- The public GitHub username entered by the user
- Public contribution dates, counts, and intensity levels returned by GitHub
- The public GitHub profile image displayed in the app
- A local snapshot used to render the app and home screen widgets

## Processing, storage, and sharing

The username and contribution snapshot are stored on the user's device. Commiturf does not operate an account server, collect analytics or advertising identifiers, request a GitHub password or access token, or sell personal information.

Requests for public contribution data and profile images are sent directly from the device to GitHub over HTTPS. GitHub may process those requests under its own privacy statement and terms. Apple and Google may separately process information when distributing the app under their respective policies.

## Retention and deletion

Local data remains until the user replaces the connected GitHub profile, chooses **Disconnect GitHub** in the profile screen, or removes the app. Disconnecting removes the saved username and contribution garden from the device and replaces the home screen widget with an empty demo garden. Commiturf has no server-side copy to retain or delete.

## Security

Network requests use HTTPS. Commiturf intentionally avoids requesting passwords, GitHub access tokens, or device permissions that are not needed for its core functionality.

## Contact

Privacy questions can be sent through the support contact published on Commiturf's App Store and Google Play product pages.

# Commiturf プライバシーポリシー

最終更新日：2026年7月23日

Commiturfは、公開GitHubプロフィールに表示されているContributionを庭として可視化する独立したアプリです。CommiturfはGitHubと提携していません。

## アプリが利用するデータ

- ユーザーが入力した公開GitHubユーザー名
- GitHubが返す公開Contributionの日付、件数、強度
- アプリ内に表示する公開GitHubプロフィール画像
- アプリとホーム画面ウィジェットの描画に使う端末内スナップショット

## 処理、保存、共有

ユーザー名とContributionスナップショットはユーザーの端末に保存されます。Commiturfはアカウントサーバーを運用せず、分析情報や広告識別子を収集せず、GitHubのパスワードやアクセストークンを要求せず、個人情報を販売しません。

公開Contributionデータとプロフィール画像のリクエストは、HTTPSを使って端末からGitHubへ直接送信されます。GitHubは自社のプライバシーステートメントと利用規約に基づいてリクエストを処理する場合があります。また、AppleとGoogleは、それぞれのポリシーに基づきアプリ配布に伴う情報を処理する場合があります。

## 保存期間と削除

端末内データは、接続するGitHubプロフィールを変更する、プロフィール画面で**GitHubとの接続を解除**を選ぶ、またはアプリを削除するまで保存されます。接続を解除すると、保存済みのユーザー名とContributionの庭が端末から削除され、ホーム画面ウィジェットは空のデモガーデンに置き換わります。Commiturfには削除対象となるサーバー上のコピーはありません。

## セキュリティ

通信にはHTTPSを使用します。Commiturfは、パスワード、GitHubアクセストークン、またはコア機能に不要な端末権限を要求しない設計です。

## お問い合わせ

プライバシーに関するお問い合わせは、App StoreおよびGoogle PlayのCommiturf製品ページに掲載するサポート窓口からご連絡ください。
