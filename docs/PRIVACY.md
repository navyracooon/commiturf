# Commiturf Privacy Policy (MVP draft)

Last updated: July 22, 2026

Commiturf displays contribution activity already visible on a public GitHub profile.

## Data we process

- The GitHub username entered by the user
- Public contribution dates, counts, and intensity levels shown by GitHub
- A local snapshot used to render the app and home screen widget

## Storage and sharing

The username and contribution snapshot are stored on the user's device. The MVP does not operate an account server, does not collect passwords or GitHub access tokens, and does not sell or share personal information. Requests to load public contribution data are sent directly from the device to GitHub.

## Deletion

Removing the app deletes its locally stored Commiturf data. A future in-app disconnect action will also clear the saved profile and snapshot.

## Third-party services

GitHub processes requests according to its own privacy statement and terms. App distribution is also subject to Apple App Store or Google Play processing.
